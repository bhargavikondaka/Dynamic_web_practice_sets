# 🛒 Add to Cart

A simple beginner-friendly web application built using **HTML, JavaScript, and Bootstrap**.

Users can enter an item name and click the **Add** button to dynamically add it to the cart list.

## 🚀 Features

- Add items to the cart
- Dynamic list creation using JavaScript
- DOM Manipulation
- Simple and responsive UI

## 🛠️ Technologies Used

- HTML5
- JavaScript
- Bootstrap 4
