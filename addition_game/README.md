# ➕ Sum of Two Numbers Game

A simple and interactive math game built using **HTML, CSS, JavaScript, and Bootstrap**.

The game generates two random numbers, and the user has to calculate and enter their sum. The application instantly checks the answer and provides feedback.

---

## 🚀 Features

- 🎲 Generates random numbers automatically
- ➕ Practice addition skills
- ✅ Instant answer validation
- 🔄 Restart button to generate new questions
- 📱 Responsive and beginner-friendly design
- 🎨 Clean user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 4

```bash
git clone https://github.com/your-username/sum-of-two-numbers-game.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

---

## 🎮 How to Play

1. Two random numbers will appear on the screen.
2. Calculate their sum.
3. Enter your answer in the input field.
4. Click the **Check** button.
5. If your answer is correct:
   - A success message is displayed.
6. If your answer is incorrect:
   - You are asked to try again.
7. Click **Restart** to generate a new question.
