# 🎨 Button Maker

A simple and interactive web application built using **HTML, CSS, JavaScript, and Bootstrap** that allows users to create and customize a button in real time.

Users can change the button's background color, font color, font size, font weight, padding, and border radius using input fields and instantly see the changes applied.

---

## 🚀 Features

- 🎨 Customize button background color
- ✍️ Change font color
- 🔠 Adjust font size
- 💪 Modify font weight
- 📏 Change button padding
- 🔲 Adjust border radius
- ⚡ Real-time button styling
- 📱 Responsive design using Bootstrap

---

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/button-maker.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

---

## 🎮 How It Works

1. Enter values for:
   - Background Color
   - Font Color
   - Font Size
   - Font Weight
   - Padding
   - Border Radius

2. Click the **Apply** button.

3. The custom button updates instantly with the selected styles.
