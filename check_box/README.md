# ☑️ Dynamic Checkbox with Label

A simple and beginner-friendly web application built using **HTML, JavaScript, and Bootstrap**.

This project dynamically creates a checkbox and its corresponding label using JavaScript DOM manipulation instead of writing the elements directly in HTML.

---

## 🚀 Features

- ☑️ Create a checkbox dynamically
- 🏷️ Generate a label using JavaScript
- 🔗 Associate the label with the checkbox
- ⚡ Practice DOM manipulation
- 📱 Simple and responsive layout using Bootstrap

---

## 🛠️ Technologies Used

- HTML5
- JavaScript (ES6)
- Bootstrap 4

## 📸 How It Works

1. JavaScript creates a checkbox (`<input type="checkbox">`).
2. A label element is created dynamically.
3. The label is linked to the checkbox using the `for` attribute.
4. Both elements are added to the webpage using DOM methods.
