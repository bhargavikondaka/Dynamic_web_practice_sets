# 🎨 Color Picker

An interactive browser-based color picker that lets users change the background color of the page by clicking on color swatches. The selected color's hex code is displayed in real time.

---

## 🚀 Demo

> Open `index.html` in any modern browser — no setup or installation required.

---

## 🎮 How It Works

1. The page loads with a white background and four color swatch buttons.
2. Click any color button to instantly change the **entire page background** to that color.
3. The **hex code** of the selected color is displayed at the bottom of the card.

---

## 📁 Project Structure

```
color-picker/
├── index.html   # Main HTML layout and button structure
├── style.css    # Styling for swatches, layout, and color display
└── script.js    # Click handlers that update background and hex display
```

## 🖥️ Getting Started

### Prerequisites

No dependencies to install. Just a web browser.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/your-username/color-picker.git

# Navigate into the project folder
cd color-picker

# Open in browser
open index.html
```

Or simply double-click `index.html` to open it in your default browser.
