# Counter App

A simple interactive counter web application built with HTML, CSS, and JavaScript.

## 📁 Project Structure

```
├── index.html
├── index.css
└── script.js
```

## 🚀 Features

- **Increment** the counter by 1
- **Decrement** the counter by 1
- **Reset** the counter to 0
- Dynamic color changes:
  - 🟢 **Green** — value is positive
  - 🔴 **Red** — value is negative
  - ⚫ **Black** — value is zero

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 4.5.2

## 📦 Setup & Usage

1. Clone or download the project files.
2. Open `index.html` in any web browser.
3. No installation or build steps required.

## 🎮 How to Use

| Button   | Action                        |
|----------|-------------------------------|
| INCREASE | Adds 1 to the counter         |
| DECREASE | Subtracts 1 from the counter  |
| RESET    | Resets the counter to 0       |

## 🎨 Styling

- Background color: `#d3ed11` (yellow-green)
- Font: **Roboto** (via Google Fonts)
- Counter value font size: `75px`
- Buttons styled with rounded borders and light background
