# 🛒 Dynamic Grocery List

A simple and beginner-friendly web application built using **HTML, CSS, JavaScript, and Bootstrap**.

This project dynamically displays a grocery list using JavaScript arrays and DOM manipulation. The grocery items are created and rendered on the webpage without writing them directly in the HTML.

---

## 🚀 Features

- 🛒 Display grocery items dynamically
- 📋 Generate a list using JavaScript
- ⚡ Create HTML elements with DOM manipulation
- 🎨 Clean and simple user interface
- 📱 Responsive design using Bootstrap

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 4

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/dynamic-grocery-list.git
```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.

---

## 📸 How It Works

1. Grocery items are stored in a JavaScript array.
2. JavaScript dynamically creates:
   - Heading
   - Card container
   - Unordered list (`<ul>`)
   - List items (`<li>`)
3. The grocery list is displayed automatically when the page loads.
