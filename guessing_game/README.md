# 🎯 Guess The Number Game

A simple and interactive browser-based number guessing game built with HTML, CSS, and JavaScript. The player tries to guess a randomly generated number between 1 and 100.

---

## 🚀 Demo

> Open `index.html` in any modern browser to play instantly — no setup required.

---

## 🎮 How to Play

1. A random number between **1 and 100** is generated when the page loads.
2. Type your guess into the input field.
3. Click the **Check** button.
4. The game tells you if your guess is **Too High**, **Too Low**, or **Correct**!
5. Keep guessing until you find the right number.

## 🛠️ Built With

- **HTML5** — Page structure and layout
- **CSS3** — Styling with Google Fonts (Roboto)
- **JavaScript (Vanilla)** — Game logic
- **Bootstrap 4.5** — Responsive grid and button components

---

## ✨ Features

- Random number generated on every page load
- Instant feedback: Too High / Too Low / Correct
- Clean, responsive UI using Bootstrap
- Color-coded result messages (blue for hints, green for correct)

---

## 🖥️ Getting Started

### Prerequisites

No dependencies to install. Just a web browser.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/your-username/guess-the-number.git

# Navigate into the project folder
cd guess-the-number

# Open in browser
open index.html
```

Or simply double-click `index.html` to open it in your default browser.
