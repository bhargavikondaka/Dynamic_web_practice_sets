# 👍 Hit the Like Button

A simple interactive web application that allows users to like and unlike an image by clicking a button. The application dynamically updates the image, button text, button color, and icon color based on the like status.

---

## 📖 Project Overview

The Hit the Like Button project demonstrates how JavaScript can be used to create interactive user experiences through DOM manipulation and event handling.

When the user clicks the Like button:

- The image changes.
- The button text updates.
- The button color changes.
- The thumbs-up icon color changes.
- The like status is toggled.

## 🎯 How It Works

1. The application loads with a default image.
2. The user clicks the **Tap to Like** button.
3. JavaScript updates:
   - The image
   - Button text
   - Button color
   - Icon color
4. Clicking again reverts everything back to the original state.
