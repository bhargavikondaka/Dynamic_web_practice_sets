# 🖼️ Image Size Adjuster

A simple and interactive web application that allows users to increase or decrease the width of an image using buttons. The image size changes dynamically, and warning messages are displayed when the image reaches its maximum or minimum size limits.

---

## 🚀 Features

- Increase image width by 5px
- Decrease image width by 5px
- Display current image width
- Maximum width limit warning
- Minimum width limit warning
- Responsive design using Bootstrap
- Beginner-friendly JavaScript project

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- Bootstrap 4

## 📸 Preview

The application starts with an image width of **200px**.

- Click **+** to increase the width.
- Click **-** to decrease the width.
- Width changes by **5px** for each click.
- Warning messages appear when limits are reached.
