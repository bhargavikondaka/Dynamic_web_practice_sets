# Click Counter

A simple and interactive Click Counter web application built using **HTML**, **CSS**, **JavaScript**, and **Bootstrap**. The application increments the counter each time the button is clicked and stores the latest count using the browser's **Local Storage**.

## 🚀 Features

- Increment counter with a button click
- Displays the current click count
- Uses Local Storage to save the count
- Responsive and clean user interface
- Beginner-friendly project for learning JavaScript DOM manipulation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 4

## 📂 Project Structure

```
├── index.html
├── style.css
├── script.js
└── README.md
```

## 📸 Preview

The application displays:

- A heading showing the current click count
- A button to increment the count
- A simple responsive layout using Bootstrap

## 💡 How It Works

1. Open the application in a web browser.
2. Click the **"Click Me!"** button.
3. The displayed counter increases by one.
4. The updated value is saved in the browser's Local Storage.
