# 🎬 Movie Reviews App

A simple and beginner-friendly web application built using **HTML, CSS, JavaScript, and Bootstrap**.

This project allows users to enter a movie title and write a review. The review is then displayed dynamically on the webpage using JavaScript and DOM manipulation.

---

## 🚀 Features

- 🎬 Add movie titles
- ✍️ Write and display movie reviews
- ⚡ Dynamic content rendering using JavaScript
- 📱 Responsive design with Bootstrap
- 🧩 Beginner-friendly project

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 4
