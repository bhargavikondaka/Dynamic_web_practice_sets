# 👤 Dynamic User Profile

A simple beginner-friendly web application built using **HTML, CSS, JavaScript, and Bootstrap**.

This project dynamically displays a user's profile information, including their profile picture, name, and age, using JavaScript DOM manipulation.

---

## 🚀 Features

- 👤 Display user profile dynamically
- 🖼️ Show profile picture
- 📝 Display user name and age
- ⚡ Create HTML elements using JavaScript
- 🎨 Simple and clean user interface
- 📱 Responsive layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 4

---

## 🎮 How It Works

1. User profile details are stored in a JavaScript object.
2. JavaScript creates the required HTML elements dynamically.
3. The profile image, name, and age are displayed on the webpage.
4. A hover effect slightly enlarges the profile card for a better user experience.
