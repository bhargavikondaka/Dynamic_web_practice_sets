# 💡 Pussy cat  Application

A simple and interactive web application that demonstrates JavaScript DOM manipulation by allowing users to switch a light bulb ON and OFF. The interface updates images, button states, and status messages dynamically based on user interaction.

---

## 🚀 Features

* Toggle light bulb between ON and OFF states
* Dynamic image updates using JavaScript
* Real-time switch status indicator
* Interactive switch board design
* Responsive and beginner-friendly user interface
* Built using HTML, CSS, JavaScript, and Bootstrap

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)
* Bootstrap 4

## 🎯 How It Works

### When the ON Button is Clicked

* The bulb image changes to the glowing bulb.
* The cat image becomes fully visible.
* The switch status changes to **"Switched On"**.
* The OFF button becomes highlighted.

### When the OFF Button is Clicked

* The bulb image changes to the switched-off bulb.
* The cat image changes to a glowing-eyes image.
* The switch status changes to **"Switched Off"**.
* The ON button becomes highlighted.
