# 🎨 Random Background Color Generator

A simple and beginner-friendly web application built using **HTML, CSS, JavaScript, and Bootstrap**.

This project changes the background color of a section whenever the user clicks the **"Click Me"** button. A random color is selected from a predefined array of colors using JavaScript.

---

## 🚀 Features

- 🎨 Generate random background colors
- 🖱️ Change color with a single button click
- 📱 Responsive layout using Bootstrap
- ⚡ Interactive user experience
- 🧑‍💻 Beginner-friendly JavaScript project

---

- Bootstrap 4

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/random-background-color-generator.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

---

## 📸 How It Works

1. Click the **Click Me** button.
2. JavaScript selects a random color from an array.
3. The background color changes instantly.
4. Repeat the process to see different colors.
