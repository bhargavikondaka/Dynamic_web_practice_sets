# 🍝 Recipe Card

A simple and beginner-friendly web application built using **HTML, CSS, JavaScript, and Bootstrap**.

This project dynamically displays a recipe card with the recipe title, image, and ingredients list using JavaScript objects and DOM manipulation.

---

## 🚀 Features

- 🍝 Display recipe details dynamically
- 🖼️ Show recipe image
- 📝 Display recipe title
- 🥗 Generate ingredients list using JavaScript
- ⚡ Dynamic content rendering with DOM manipulation
- 📱 Responsive design using Bootstrap

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 4

---

## 🍽️ How It Works

1. Recipe details are stored inside a JavaScript object.
2. The application dynamically displays:
   - Recipe title
   - Recipe image
   - Ingredients list
3. JavaScript creates the ingredients list and renders it on the webpage using DOM methods.
