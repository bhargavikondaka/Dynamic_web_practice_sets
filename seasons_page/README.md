# 🌸 Seasons Switcher

A simple and interactive web application that allows users to switch between different seasons (Spring, Summer, Autumn, and Winter) by clicking buttons. The displayed image changes dynamically based on the selected season.

## 📖 Project Overview

The Seasons Switcher project demonstrates DOM manipulation using JavaScript. Users can click season buttons to view corresponding seasonal images.

This project helps in understanding:

- DOM Manipulation
- Event Handling

- Responsive Images

## 📸 How It Works

1. The application displays a default season image.
2. Users click one of the season buttons.
3. JavaScript updates the image source dynamically.
4. The selected season image is displayed instantly.
