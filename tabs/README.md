# 🏛️ Varanasi Tourism Tabs

A simple and interactive web application built using **HTML, CSS, JavaScript, and Bootstrap**.

This project displays information about **Varanasi** using a tab-based interface. Users can switch between different sections such as **About**, **Best Time to Visit**, and **Attractions** without reloading the page.

---

## 🚀 Features

- 🏛️ Learn about Varanasi
- 📅 View the best time to visit
- 📍 Explore popular attractions
- 🖱️ Interactive tab navigation
- ⚡ Dynamic content switching using JavaScript
- 📱 Responsive design with Bootstrap

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 4

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/varanasi-tourism-tabs.git
```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.

---

## 🎮 How It Works

1. Open the application.
2. Click on the **About** tab to learn about Varanasi.
3. Click **Time to Visit** to view the ideal season for visiting the city.
4. Click **Attractions** to explore some of the famous places in Varanasi.
5. The content changes instantly without refreshing the page.
