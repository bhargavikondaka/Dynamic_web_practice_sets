# 💰 Tip Calculator

A simple and beginner-friendly **Tip Calculator** built using **HTML, CSS, and JavaScript**.

This project helps users calculate the tip amount and total bill amount based on the bill value and tip percentage entered by the user.

---

🌐 [View Live Project]

## ✨ Features

- Enter bill amount
- Enter tip percentage
- Automatically calculate tip amount
- Calculate total amount including tip
- Input validation with error messages
- Simple and user-friendly interface

## 📸 How It Works

1. Enter the bill amount.
2. Enter the tip percentage.
3. Click the **Calculate** button.
4. The application displays:
   - Tip Amount
   - Total Amount
