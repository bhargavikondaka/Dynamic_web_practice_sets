# ✅ Dynamic Todo List

A simple and beginner-friendly **Todo List** web application built using **HTML, CSS, JavaScript, Bootstrap, and Font Awesome**.

This project dynamically displays a list of tasks using JavaScript objects and DOM manipulation. Each task includes a checkbox and a delete icon, providing the foundation for a basic task management application.

---

## 🚀 Features

- 📝 Display todo items dynamically
- ☑️ Checkbox for each task
- 🗑️ Delete icon for every todo item
- ⚡ Dynamic DOM element creation
- 📱 Responsive layout using Bootstrap
- 🎨 Clean and user-friendly interface

---

## 📸 How It Works

1. Todo items are stored in a JavaScript array of objects.
2. JavaScript dynamically creates:
   - Checkbox
   - Task label
   - Delete icon
3. Each todo item is rendered on the webpage using DOM manipulation.
4. The layout is styled using Bootstrap and custom CSS.
