# 🚦 Traffic Light Project

A simple **Traffic Light Simulation** built using **HTML, CSS, and JavaScript**.

This project allows users to control a traffic light by clicking the **Stop**, **Ready**, and **Go** buttons. When a button is clicked, the corresponding traffic light color turns on while the other lights turn off.

---

## 📌 Features

- 🔴 Stop button turns on the red light
- 🟡 Ready button turns on the yellow light
- 🟢 Go button turns on the green light
- 🎨 Simple and responsive user interface
- ⚡ Interactive functionality using JavaScript

---
